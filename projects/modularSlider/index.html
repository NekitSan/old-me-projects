Version sliders - 1.0.0

Описание:

Сладеры реализованы через работу с DOM. Сделал так, что пользователю не нужно задумываться о кол-ва слайдов, вы настраиваете или используете стили по пумолчанию. Для меня как для разработчика, недостатком является, скорее всего, добавление картинок в html документ, хотя думаю сделаю доп. слайдер использующий JSON-файл, в котором будут лежать url, alt - слайдов(картинков). Вообще эту тему со слайдерами, можно вертеть как угодно, но по простате и логике, мне кажеться, целесобразнее использовать именно такой способ. Хотя бы потому, если приходят картинки со сервера в html разметку, то разработчику не нужно задумыватсья как их обрабатывать, за него все делает скрип.

На чисто JavaScript.
